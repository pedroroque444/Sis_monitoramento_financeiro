# FlexSpendeR

## Tecnologias e Arquitetura
- **Linguagem:** Python
- **Interface Gráfica:** CustomTkinter 
- **Análise de Dados:** Matplotlib 
- **Persistência:** SQLite3 
- **Padrão Estrutural:** Arquitetura modular com separação estrita de camadas (GUI, Controle e Banco de Dados)

## Como Executar o Programa

### Opção 1: Executável Direto (Recomendado para Avaliação)
Para testar o sistema imediatamente sem a necessidade de instalar o Python ou dependências técnicas na sua máquina:
1. Vá até a seção **Releases** deste repositório (na coluna lateral direita).
2. Baixe o arquivo executável `.exe`.
3. Execute o aplicativo no seu ambiente Windows.

### Opção 2: Executando via Código-Fonte
Caso deseje rodar o projeto a partir dos scripts originais, certifique-se de ter o Python instalado e execute os comandos abaixo no terminal:

```bash
# Instalar as dependências necessárias
pip install customtkinter matplotlib

# Executar a aplicação
python main.py
